# AI Agent Images - Manual Setup Required

The AI Squad section expects 8 agent images in PNG format. Due to access restrictions, these images need to be manually downloaded and saved.

Please save the following images from your GitHub comment to this directory (`public/images/agents/`):

## How to Save the Images

1. Right-click on each image in your GitHub comment
2. Save as PNG format with the exact filename listed above
3. Place all 8 images in this directory: `public/images/agents/`

## Expected File Structure

```
public/images/agents/
├── blaze.png
├── echo.png
├── glitch.png
├── lexi.png
├── lumi.png
├── nova.png
├── roxy.png
└── vex.png
```

Once all 8 images are saved, the AI Squad section will display your original agent avatars as intended.
